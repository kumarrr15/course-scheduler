# Course Scheduler Application

A Java-based course scheduling system that allows administrators to manage semesters,
courses, and classes, and allows students to schedule or drop classes.

## Technologies
- Java
- JDBC
- Apache Derby
- Swing (NetBeans)

## Features
- Admin and Student user roles
- Course scheduling with seat limits
- Queue-based waitlist management
- Automatic enrollment when seats become available
- Database-driven design using prepared SQL statements

## Running the Project
- Open the project in NetBeans
- Ensure Apache Derby is available
- Run the main GUI class
